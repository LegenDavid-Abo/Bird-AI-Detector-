Dual-Stage AI Architecture

01

Object Detection (COCO-SSD)

Single Shot MultiBox Detector locates birds with bounding boxes. Trained on 90 object classes including birds, with real-time performance optimized for browser deployment.

02

Species Classification (MobileNetV2)

Lightweight deep CNN classifies detected birds into species. Uses inverted residuals and linear bottlenecks for efficient feature extraction with 3.5M parameters.

03

Ensemble Validation

Cross-validation between detection confidence and classification probability ensures high-accuracy results, filtering false positives through multi-model consensus.
